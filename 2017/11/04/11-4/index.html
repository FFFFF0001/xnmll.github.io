考研 大学

发表于 2017-11-04 | 分类于 感悟 | | 本文总阅读量 次
字数统计: 877 | 阅读时长 ≈ 3

前言:

写作时间：2017-11-04 大三上学期星期六的下午。

我想，我目前是很平静的做了人生中的一个重要决定，就是，下定决心准备考研。前几天在一部美剧中有看到这样的台词 生活就是二进制，一个个0101组成的。每一次抉择，选了就是1，不选就是0。 说实话，我在以前的人生道路上选择了不少的0，儿时并没有学到很多才艺，至少会一门乐器；高中的重要关头沉迷网游，高考失意… 现在站在分叉路口的我并不再这么迷茫，我仿佛可以看见自己在以后的一年里，10年里做着自己坚定的事，在未来的某一天成为行业大牛。

WHAT:

1.什么是考研？规模，意义，重要性？

考研就是 全国硕士研究生统一招生考试的简称，具体在我的身上就是：难度系数高，意义重大的人生中最后一次统一考试。

2.应该怎么准备？

科目是：政治，英语，数一，专业课

英语数学和专业课是需要长期下功夫的，就目前的英语，数学水平来说，这将会是一场艰难的持久战。目前到12月的四级考试每天背一个list单词，最后几天每天复习4个list及做真题，先把12月的四级完成了。四级之后的时间开始准备数学，每天看数学及背单词加专业课，思路状态都调整差不多后狂刷19级资料书。下学期考六级。每天的时间安排有序，不做打游戏，看直播等浪费时间的事。大概安排就是这样。

WHY:

1.自我挑战

在之前的某一段时期，如果你问我是否要考研，我一定会毅然决然的说：NO。我相信学历是三本的技术大牛当然可以闯出一片天，我也更害怕自己在努力一年多以后考研失败，浪费了一年的时光，如果没有选择考研，自己会在专业方面离大牛更近一步，但是如果考研失败，自己将面临很尴尬的处境。不得不说，这是一场自我挑战。

2.新的高度

但是，就我自身的条件来说，没有在专业上有一个很好的基础，还是很难走的很远，程序=算法+数据结构，以后面对的真正的项目会比我想像的复杂很多，并不是现在简简单单的demo所能比拟。考研会把我带到新的高度上去，让我在这条路走的长远。

3.人生遗憾

高考是我人生中的遗憾，也希望考研成功可以稍微弥补一点。

HOW:

1.时间及任务

时间还有一年多一个月，还不算紧凑。但是自己的基础不好，任务极重。

2.calendar

在以后的两个星期：

6:00-7:30 12:10-1:30 6:10-9:30
一个list单词+10个词根 复习单词 专业课

变动再补充。

3.心态

不崩就好…

最后:

在新的博客上 第一次写这么多…

这篇文章竟然用了我近一个小时的时间！嗯，相信未来充满希翼，努力成为行业大牛，早日给父母养老，不为之前做的决定而后悔。

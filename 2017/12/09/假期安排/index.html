假期安排

发表于 2017-12-09 | 分类于 计划 | | 本文总阅读量 次
字数统计: 239 | 阅读时长 ≈ 1

这个周六日常的没有完成任务，写下假期安排吧，明早早起..

假期大概55天左右，去掉年中，算是50天

1.总任务及安排

额，有次在日记中整理了，先写上点以后补充

1.考研英语（绿皮第一遍，第二遍）

2.数学（看教学视频，起码过一遍）

3.专业课（了解后写内容）

4.政治了解

5.预习大三下学期所有课程，编程类提前看书写代码等，所有编程类学科尽量下学期不用上课去做更有意义的事

6.每天半小时以上看行业资讯，思考每天所学

2.时间

6点起 上午下午各两个档和学校类似 晚上加两个档。

大概就是这样

3.important

18周以后就算是假期了，时间按照假期安排

两周课程设计不能浪费时间，全力以赴
